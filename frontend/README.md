# Data Explorer — frontend

React + TypeScript + Vite SPA. See the repository root
[`README.md`](../README.md) for the full project overview and
[`docs/DEVELOPER_GUIDE.md`](../docs/DEVELOPER_GUIDE.md) for setup and
conventions.

```bash
npm install
npm run dev      # http://localhost:5173, proxies /api/v1 to :8080
npm run build
npm run lint
npx tsc -b
```
